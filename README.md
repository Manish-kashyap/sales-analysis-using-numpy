# Company Sales Analysis using NumPy

A simple Python project that analyzes a company's monthly sales data using the **NumPy** library. The program accepts sales figures for all 12 months, performs statistical analysis, and identifies key business insights such as total sales, average monthly sales, highest and lowest sales months, and months performing above or below the annual average.

## Features

* Accepts monthly sales input from the user.
* Calculates total annual sales.
* Computes average monthly sales.
* Identifies the month with the highest sales.
* Identifies the month with the lowest sales.
* Displays months with above-average sales.
* Displays months with below-average sales.
* Demonstrates the use of NumPy functions such as `array()`, `sum()`, `mean()`, `argmax()`, `argmin()`, `max()`, and `min()`.

## Technologies Used

* Python 3
* NumPy

## Learning Objectives

This project helps beginners understand:

* NumPy arrays
* User input handling
* Statistical calculations
* Boolean indexing
* Finding maximum and minimum values
* Data analysis fundamentals using Python
